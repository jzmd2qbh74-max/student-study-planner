# Student Study Planner

A full-stack web application for students to manage courses, assignments, exams, study sessions, and grades in one place.

## Tech Stack

- **Backend:** Java 21, Spring Boot, Spring Data JPA, Spring Security (JWT)
- **Database:** MySQL
- **Frontend:** React + TypeScript (planned)
- **Tools:** Docker, Maven, Postman

## Status

🚧 In development — currently building out core domain entities and database schema.

## Features (planned)

- User registration & login
- Course, assignment, and exam management
- Study session planner
- Grade tracking
- Calendar view
- Notifications/reminders
